# ABSTRACT
WeatherSky is a lightweight, client-side weather dashboard that lets users search any city or use their current location to view real-time conditions and a 5‑day forecast. The interface emphasizes clarity and visual appeal with glassmorphism styling, animated icons, and responsive layout for mobile and desktop.

# INTRODUCTION
This project is a single‑page web app built with HTML, CSS, and JavaScript. It integrates a geocoding service to resolve city names to coordinates and then fetches forecast data from a public weather API. A theme toggle and dynamic background gradients adapt the UI to conditions.

# OBJECTIVE
Provide a simple, fast, and visually engaging way for users to check current weather, humidity, wind speed, and upcoming daily forecasts without installing an app.

# ADVANTAGES
- Simple UI with quick search and current‑location support
- Real‑time data with a clear 5‑day forecast view
- Responsive design with accessible typography and icons
- Theme toggle and dynamic backgrounds improve readability

# DISADVANTAGES
- Requires an active internet connection to fetch API data
- Accuracy depends on third‑party API availability and limits
- Basic feature set (no hourly forecast, alerts, or saved locations)

# CONCLUSION
WeatherSky demonstrates how modern web technologies can deliver a polished, usable weather experience with minimal setup. It is a strong foundation for expanding features such as saved cities, hourly charts, and severe‑weather notifications.
